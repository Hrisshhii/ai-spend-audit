# AI Spend Audit

An AI SaaS cost optimization platform that analyzes AI tool subscriptions and recommends cheaper alternatives.

## Features

- Multi-tool AI spend tracking
- AI-generated optimization summaries
- Shared audit pages
- Lead capture form
- Dynamic savings calculations
- Supabase integration
- Zustand state persistence

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase
- Zustand
- Vercel
